# @eduzz/houston-mobile

Eduzz Houston Mobile Components

## Installation

```sh
yarn add @eduzz/houston-mobile
```

## Usage

```js
import { Modal } from '@eduzz/houston-mobile';

// ...

<Modal visible={visible} onRequestClose={closeModal}>
  ...
</Modal>
```

## Examples

```sh
# Android app
yarn example android
# iOS app
yarn example ios
```
